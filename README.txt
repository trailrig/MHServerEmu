MHServerEmu - Version 0.2.0

MHServerEmu is licensed under AGPLv3. The source code is available on GitHub: https://github.com/Crypto137/MHServerEmu.

	Setup
1. Make sure you have .NET Desktop Runtime 6 installed. You can download it here: https://dotnet.microsoft.com/en-us/download/dotnet/6.0.
2. Acquire a copy of Marvel Heroes Omega, version 1.52.0.1700. This is the version of the game that you can download from Steam if you still have the game in your library.
3. Find where your game files are located. You can do that by right clicking on the game in your Steam library and choosing Manage -> Browse local files.
4. Run the included SetupSorcererSupreme tool and follow the instructions.

	Running
1. Run the included StartServers.bat file and wait for MHServerEmu to initialize.
2. (Optional) Open http://localhost:8080/AccountManagement/Create in your web browser and create an account. Note: this link is going to work only when the servers are running.
3. Run StartClient.bat and log in with your created account OR run StartClientAutoLogin.bat to play with a default account.
4. When you are done, run StopServers.bat to stop the servers.

Have fun!
